# Gen-AI Log Analysis & Monitoring Dashboard

## Overview
An end-to-end log analysis and monitoring system designed to detect system failures, identify root causes, and visualize real-time insights through an interactive dashboard. The system processes large volumes of log data and presents actionable insights for efficient debugging and system monitoring.

## Live Demo
https://9059963940.github.io/genai-log-analyzer/

## Features
- Real-time log monitoring and visualization
- Anomaly detection using NLP-based analysis
- Interactive dashboard for system insights
- Dynamic UI updates using data-driven rendering
- Failure pattern identification and root cause analysis
- Scalable architecture for handling large log data

## Tech Stack
- Frontend: Streamlit (Interactive UI)
- Backend: Python
- Data Processing: NLP, Log Parsing
- APIs: REST-based data flow
- Data Format: JSON

## Architecture
- Log data ingestion and preprocessing
- NLP-based anomaly detection engine
- Backend processing pipeline
- REST API integration for data flow
- Frontend dashboard for visualization and monitoring
